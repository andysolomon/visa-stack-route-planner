# visa-stack-route-planner

Route planning scaffold for long-term travelers who need visa-compliant itineraries across multiple countries.

## Stack
- Next.js (App Router) + TypeScript
- Bun
- Tailwind CSS

## Getting Started
1. Install dependencies: `bun install`
2. Copy env file: `cp .env.example .env.local`
3. Run dev server: `bun run dev`

## Scripts
- `bun run dev`
- `bun run build`
- `bun run start`
- `bun run lint`
- `bun run typecheck`
- `bun run test`

## Notes
- Includes scaffold docs: `IMPLEMENTATION_PLAN.md` and `progress.txt`
